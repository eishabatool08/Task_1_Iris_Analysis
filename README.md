#Iris Flower Classification & Analysis 🌸

#Project Overview
This project performs an Exploratory Data Analysis (EDA) and builds a classification model to identify three species of the Iris flower: **Setosa, Versicolor, and Virginica**.

#Key Features
- **Exploratory Data Analysis:** Visualizing feature distributions using Seaborn and Matplotlib.
- **Data Visualization:** Pairplots and Heatmaps were used to identify correlations between sepal and petal measurements.
- **Machine Learning:** Implemented classification to predict species based on floral dimensions.

#Libraries Used
- **Pandas** for data manipulation.
- **Seaborn & Matplotlib** for advanced visualizations.
- **Scikit-learn** for machine learning algorithms.

#How to Run
1. Clone this repository.
2. Ensure you have Jupyter Notebook installed.
3. Open `Task_1_Iris_Analysis.ipynb` and run all cells.
